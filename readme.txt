Я выбрал для хакатона проект Log_parser
Этот проект главным образом направлен на работу со строками и массивами.
При запуске в качестве входного параметра необходимо передать имя файла. В директории приложения находится файл testlog.log, который для этого подойдет. 
Программа ищет все методы, вызовы которых были зафиксированы в логи, считает количество завершенных вызовов, максимальное, минимальное и среднее время выполнения этих методов. Результаты выводятся на консоль. 
Прошу запускать с IDEA
